# Multi Step Form With React

[You can see screenshots of the messenger below](#screenshots)

## Install

1. `npm install` или `yarn install`
2. Run command `npm start` or `yarn start`.

## Build with:

**Frontend:**

- ReactJS + hooks
- Material-UI
- React-hooks-helper
